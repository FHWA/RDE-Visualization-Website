RDE DATA VISUALIZATION

Exploring The RDE

A wealth of transportation data at your fingertips

The Research Data Exchange contains a wide variety of data sets covering many cities across the United States. Each data set is associated with a parent data environment and has been tagged by data type, frequency, facility, and location. Without the ability to browse data sets by each of these characteristics, the task of finding the right data set can be daunting.

This visualization allows users to explore the full list of data sets by simply clicking on characteristics of interest. For each chart, clicking on a value filters all other charts on that value. This powerful, dynamic drill-down capability enables users to quickly identify data sets of interest and jump straight into research.

Example

Try clicking on the bar labeled 'basic safety message (bsm)' on the chart labeled 'Number of Data Sets by Data Type'. All other charts will automatically filter to display only data sets with Basic Safety Message (BSM) data. Multiple filters can be applied at once; clicking the 'arterial' bar in the chart labeled 'Number of Data Sets by Facility' further narrows the selection to only data sets containing BSM data recorded on arterials.

Using the Dashboard

Hover over any characteristic to see how many data sets matching the current set of filters have that characteristic. Filters can be reset by clicking the 'Clear Filter' link that appears on a chart when a filter is applied or clicking on the characteristic that was selected to apply the filter.

Notes

Data sets can have more than one Data Type, Facility, or Data Frequency tag. Therefore, each data set can be represented more than once in those charts.

The 'Number of Active Data Sets Over Time' chart displays the following: for each time point, count the number of data sets where the time point is between that data set's start date and end data (hence, 'Active Data Sets').
